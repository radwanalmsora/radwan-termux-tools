# Radwan Termux Tools 🚀

Powerful tools for Termux users on Android.

## Features
- Device Info
- IP Lookup
- Hash Generator
- Website Scanner

## Installation

```bash
pkg update && pkg upgrade -y
pkg install git -y
git clone https://github.com/YOUR_USERNAME/radwan-termux-tools.git
cd radwan-termux-tools
chmod +x *
mv radwan-* /data/data/com.termux/files/usr/bin/
